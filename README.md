This is a college project to demonstrate and practise CI/CD concepts in a practical application.

This is a demo web application that users can use to create, store and sign petitions online.
It's utilizes github webhooks, a Jenkins pipeline, and a Tomcat server to allow immediate testing and deployment of new project builds.
(Jenkins and Tomcat are hosted on a AWS EC2 instance, which has not been kept running after the projects conclusion)
